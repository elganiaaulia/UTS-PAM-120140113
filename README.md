# Build
Aplikasi pricelist makeup ini dapat di build dengan cara Expo CLI dan eas-cli.

npm install -g expo-cli
npm install -g eas-cli
Kemudian, Anda dapat membangun aplikasi dengan perintah berikut.

eas build
